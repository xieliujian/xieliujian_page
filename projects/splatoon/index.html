项目介绍

本 Demo 使用 Unity 5 还原了任天堂主机游戏《喷射战士（Splatoon）》的核心染色机制。 玩家发射墨水时，通过 RenderTexture + Shader 将颜色实时绘制到场景地面与物体表面， 支持多颜色混合与遮罩检测，模型表面的染色范围可动态扩展。


核心技术点：世界空间 UV 投影、RenderTexture 动态绘制、 自定义 Surface Shader 读取墨水遮罩，以及碰撞检测触发绘制逻辑。 项目已在 GitHub 获得 74 Stars，并附有知乎技术分享文章。
